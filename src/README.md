Проект представляет собой высокоуровневый автоматизированный фреймворк для тестирования функционала
и пользовательского интерфейса крупнейшего сервиса доставки продуктов в Беларуси — edostavka.by. 

Окружение
Язык программирования: Java 21
Сборка проекта: Maven
Браузер: Google Chrome (последняя стабильная версия)

Технологический стек:
* **Язык разработки:** Java 21
* **Сборщик проекта:** Maven
* **Тестирование UI:** Selenium WebDriver 4.44.0
* **Тестирование API:** REST Assured 6.0.0
* **Генерация тестовых данных:** Datafaker 2.5.4
* **Тестовый фреймворк:** JUnit5 5.11.3
* **Логирование:** Log4j2 2.24.3
* **Отчетность:** Allure Report 2.29.1
* **Непрерывная интеграция:** Jenkins

Паттерны проектирования и архитектурные решения

Page Object Model (POM) - каждая страница представлена отдельным классом, который содержит локаторы 
и методы взаимодействия UI-элементов.

Singleton - Управление WebDriver реализовано на основе паттерна Singleton, что гарантирует использование 
единого экземпляра драйвера в рамках всего тестирования
